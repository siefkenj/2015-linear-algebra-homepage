Latex

Latex is a programming language designed for typesetting documents. It was created (technically the base layer called Tex) by Donald Knuth (considered one of the fathers of computer science).

When you author a document in Latex, you write the content in a text file (ending in .tex) and then compile to a pdf. The compiler lays out all the words on the page and ensures all spacing, etc. is consistent. This means on the one hand that you won't see what your document will look like until you compile it, but you also will not need to do as much tweaking (as you might in a program like Microsoft Word) since the automatic layout algorithm latex uses is quite good (your textbook is written in Latex).

Getting Latex

You can either use an online version of Latex (which requires no installations), or you can install Latex on your computer. I recommend starting with the online Latex distribution provided by ShareLaTeX https://www.sharelatex.com/. Sign up for a free account and get started!

If you want Latex installed locally on your computer, you can install one of the Tex distributions for your operating system. For Windows, I recommend MiKTeX; for Linux TeX Live (which should be available through your package manager); and for Mac Mac Tex.
